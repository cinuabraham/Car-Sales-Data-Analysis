🚗 Car Sales Data Transformation & Insights

In this project, I worked on transforming large datasets into a clean, structured and insight-ready format. I developed and utilized master table using SQL joins to consolidate data into a single, efficient process for analysis.

This approach enabled me to evaluate key business metrics, including Total cars, Total sales, Total brands, Total fuel types and car models. I also analysed sales trends over time, compared manual vs automatic transmission preferences and explored sales performance by year to identify patterns and opportunities.
Tools & Skills Used:

⚙️SQL Server (SSMS)
⚙️Data Cleaning and Modeling
⚙️Dashboard Design and Story Telling


This project enabled me to strengthen my skills in 
✅Cleaned and pre-processed raw datasets in Excel to ensure data accuracy, consistency, and readiness for analysis.
✅ Executed a comprehensive ETL (Extract, Transform, Load) process to efficiently structure and prepare data for analysis
✅ Imported and managed the data seamlessly in SQL Server
✅ Built master table using SQL joins to create a unified data model
✅ Developed interactive and professional dashboards in Power BI driven by SQL queries
